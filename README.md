描述
===========================
:bullettrain_front: `saltire-cli`是一款简单易用的脚手架工具，你只需要记住它的两个 API 便可以快速上手，其中一个用于列举目前存在的模板，另一个则用于创建项目

****


API | 说明
---|---
list | 列举所有可用的模板
<模板名称> <项目名称> | 使用指定的模板创建指定的项目


### 可用模板

模板名称 | 模板说明
---|---
right-pc | 快速搭建骑士 pc 端站点
ex | 快速搭建个人测试站点

### 使用案例 1 - 列举可用模板

```
saltire-cli list
```

### 使用案例 2 - 创建个人测试站点

```
saltire-cli ex exnpm
```
